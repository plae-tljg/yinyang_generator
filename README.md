# 阴阳怪气转换器 (YinYang Converter)

万物负阴而抱阳，冲气以为和

一个有趣的文字转换工具，将普通文字转换成阴阳怪气的语气。

## 项目结构

```
yinyang_site/
├── index.html          # 主 HTML 文件
├── README.md           # 项目说明
└── public/             # 公共资源文件夹
    ├── css/
    │   └── style.css   # 样式文件
    └── js/
        └── app.js      # JavaScript 逻辑文件
```

## 功能特性

- ✨ 智能文字转换：将普通文字转换成阴阳怪气的语气
- 🎭 多种转换模板：随机选择不同的转换方式
- 😏 表情替换：支持多种表情符号
- 📋 一键复制：快速复制转换结果
- ⌨️ 键盘快捷键：支持 Ctrl/Cmd + Enter 快速转换
- 📱 响应式设计：适配移动端和桌面端

## 使用方法

1. 在输入框中输入要转换的文字
2. 点击"转换阴阳怪气"按钮或使用 Ctrl/Cmd + Enter
3. 查看转换结果
4. 点击表情按钮可以替换表情
5. 点击"复制结果"按钮复制转换后的文字

## GitHub Pages 部署

1. 将代码推送到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择主分支作为源
4. 访问 `https://plae-tljg.github.io/yinyang_generator` 即可

## 技术栈

- 纯 HTML/CSS/JavaScript（Vanilla JS）
- Font Awesome 图标库（CDN）
- 无依赖，可直接在浏览器中运行
